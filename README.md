# Diagramador de Flujo Pro 🚀

[![Ko-fi](https://img.shields.io/badge/Ko--fi-Invítame%20un%20café-29abe2?style=for-the-badge&logo=ko-fi&logoColor=white)](https://ko-fi.com/brokercl)

**Diagramador de Flujo Pro** es una herramienta de escritorio ligera, nativa y ultrarrápida, diseñada específicamente para crear diagramas de arquitectura, flujos lógicos y algoritmos sin distracciones en macOS.

Olvídate de las pesadas aplicaciones web o de pagar suscripciones mensuales. Esta es una herramienta pensada por y para desarrolladores, arquitectos de software y analistas de sistemas que valoran el rendimiento, la independencia y la privacidad.

---

## ✨ Características Principales

* **Lienzo Infinito (360°):** Navega libremente por tu espacio de trabajo arrastrando el fondo con el clic derecho del ratón.
* **Motor de Texto Premium:** Edición de texto fluida con soporte para selección precisa de caracteres (clic + arrastre), cursor interactivo parpadeante, navegación mediante flechas del teclado y soporte completo para atajos nativos (`Cmd+A`, `Cmd+C`, `Cmd+V`) conectados al portapapeles del sistema.
* **Interfaz "Accordion" Responsiva:** Menú lateral elegante, colapsable y escalable que maximiza tu área de trabajo cuando lo necesitas.
* **Lógica de Conexiones Inteligentes:** Nodos magnéticos con múltiples puntos de anclaje, flechas elásticas y redimensionamiento dinámico en tiempo real.
* **Formatos Locales:** Guarda y carga tus proyectos de manera nativa sin perder la configuración de colores ni las dependencias jerárquicas.
* **Ultra Ligero:** Empaquetado en un instalador optimizado de apenas ~14 MB que no requiere conexión a internet para funcionar.

## 📥 Instalación

Para descargar la última versión estable del instalador nativo, dirígete a la sección de **[Releases (Lanzamientos)](https://github.com/brokercl/diagramador-de-flujo-pro/releases/tag/v1.0.0)** de este repositorio.

1. Descarga el archivo `Diagramador_de_Flujo_Pro.dmg`.
2. Haz doble clic para montarlo y arrastra la aplicación **Diagramador de Flujo Pro** a tu carpeta de **Aplicaciones**.
3. **⚠️ Nota de Seguridad de macOS (Gatekeeper):** Al ser una aplicación independiente de código abierto, la primera vez que la intentes ejecutar macOS podría retenerla. Para abrirla sin problemas:
   * Ve a tu carpeta de **Aplicaciones** en el Finder.
   * Haz **Clic Derecho** (o `Control + Clic`) sobre la aplicación.
   * Selecciona **"Abrir"** en el menú contextual.
   * En la ventana de advertencia, haz clic en **"Abrir de todos modos"** (este paso solo es necesario la primera vez).
